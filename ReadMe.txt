In order to start application you need:
Tech stack, that was used: MS SQL, ASP.NET CORE, Angular.
1)change in the application.json SignalRBackend folder DefaultConnection "Server=DESKTOP-A8FRI08;Database=chatdb;Trusted_Connection=True;" to your db connection;
2) start command npm install in the SignalFrontend project
3) start Backend project and then start FrontEnd project (npm start command).

You can swtich between chats, send messages and do other actions with messages with the help of right click.
